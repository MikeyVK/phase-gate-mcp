# Epic: Installable Wheel / Standalone MCP Server

## Problem
De MCP server is een lokaal projectspecifiek script (start_mcp_server.ps1) zonder gestandaardiseerd installatiepad. Distributie naar andere projecten vereist handmatige kopiëring van de volledige mcp_server/ directory en handmatige aanpassing van paden. De horizon is deployment als herbruikbare tool in andere projecten.

## Expected Behavior

De MCP server is installeerbaar als Python wheel met een console_scripts entry point (st3-mcp). Eerste keer opstarten bootstrapt automatisch de default .st3/ structuur. De directorynaam is configureerbaar zodat de server in elk project gebruikt kan worden.
## Actual Behavior

Nog niet gerealiseerd. Vereiste stappen: (1) MCPServer composition root scheiden van runtime dispatch (#285), (2) .st3/ root configureerbaar maken (#260), (3) console_scripts entry point in pyproject.toml declareren, (4) bootstrap-logica aanmaken bij first run.
## Context

Ontstaan uit backlog review #288 (2026-04-24). De 'stip op de horizon' voor dit project is deployment van de MCP server als herbruikbare tool in andere projecten. Child issues: #285, #260 + 2 nieuw aan te maken issues.
